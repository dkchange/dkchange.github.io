May 13, 2020

測試資料排列組合

目前有個案例是要衣服要件所有可能的排列組合 輸入範例：

{
'color': ['red','white','blue'],
'size':['s','m','l','xl'],
}

希望結果能輸出 3*4 種排列組合

爾且增加衣服的要件依然能運作，例如新增衣服主題 subject

{
'color': ['red','white','blue'],
'size':['s','m','l','xl'],
'subject':['cartoon','aphabet','none']
}

希望結果能輸出 343 種排列組合 輸出範例：

{
    [
        'color':'red'
        'size':'s'
        'subject':'cartoon'
    ],
    [
        'color':'white'
        'size':'s'
        'subject':'cartoon'
    ]
    ...
}
