Q:用最簡單的PHP代碼來實現簡單工廠模式

interface Car {
    public function drive();
}

class Tesla implements Car {
    public function drive() {
        echo "Driving a Tesla car!\n";
    }
}

class BMW implements Car {
    public function drive() {
        echo "Driving a BMW car!\n";
    }
}

class CarFactory {
    public static function createCar($type) {
        if ($type == 'Tesla') {
            return new Tesla();
        } else if ($type == 'BMW') {
            return new BMW();
        } else {
            throw new Exception("Invalid car type.");
        }
    }
}

$tesla = CarFactory::createCar('Tesla');
$tesla->drive();

$bmw = CarFactory::createCar('BMW');
$bmw->drive();

從上面方法我們看出需要interface定義相同類型的產品 需要class的對應來判斷要news哪個class，而這個判斷就是createCar方法
