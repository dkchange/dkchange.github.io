效能分析

這邊蠻重視數學的
log？ n 次方程式？

  • 時間複雜度
    編譯時間+input 所影響的時間
    一般來講就是程式預計要走幾步

    • 大寫的英文字母 O (big oh)函數，代表演算法執行步驟數目上限；
    • 大寫的希臘字母 Ω (Omega)函數，代表下限；
    • 大寫的希臘字母 Θ (Theta)函數，代表同時滿足上限與下限
      那我們通常用的是 O
      f(n)<=cg(n)

  • 空間複雜度(佔用多少記憶體)
    code 的大小+input 的量所影響的空間

陣列

定義：一組索引(index)和值(value)的集合
資料結構：一個索引對應一個值
實作：在電腦裡面用連續空間來存放
應用例子：矩陣、多項式、字串

stack: a last in first out list

定義：有限的元素
實作：用陣列來實作
應用例子：老鼠走迷宮

queue: a first in first out list

定義：有限的元素 多了兩個變數指向 font 前面 rear 後面
實作：用陣列來實作
應用例子：job scheduling

circular queue

定義：實質上還是一長列的 queue
實作：用 queue 實現
好處：充分利用空間
